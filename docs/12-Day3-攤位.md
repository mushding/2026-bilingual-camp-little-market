# Day3 攤位與結關（石門國小 09:35–10:35）

> Day3 攤位 = **Day2 全套（B-1 ~ B-18，含郵政感謝卡核銷）** + **強化天國攤**。
> 重點差異：① 強化天國攤（交帳主題）② 10:25 市場關閉突襲 ③ 結關交帳 ④ 理財成績單發放。
> 所有數字以《小市集單一真相源 SOT v1.0》為唯一來源。

## 2. 強化天國攤（Day3 限定，扣合「交帳」）

> KP＝天國點數（管家獎），**不可換現金、不計入積分榜**，獨立第二軌。
> 以下三項為 Day3 在原奉獻／聽見證／感謝卡（郵政核銷，加給寄件人）之上**加開**的橋段。

### 卡片 2-1：奉獻交帳挑戰（強化版舊鞋救命）

| 欄位 | 內容 |
|---|---|
| 關卡介紹 | 在原「現金 1:1 轉 KP」之上，Day3 加開：單筆奉獻 ≥100，額外一次性 **+50 KP bonus**（每生限一次）。 |
| 開關詞 | 「最後一天交帳了！今天奉獻 100 以上，除了 1 比 1 換成天國點數，再多送你 50 點。天國點數是唯一帶得走的，把握機會。」 |
| 道具與數量 | 舊鞋救命海報 ×1（加大）、木板「舊鞋救命」×1 |
| 關主人數 | 1 人（加派宣傳） |
| 準備時間 | 0（沿用 Day2 攤）|
| 總時長（單次） | 1–2 分 |
| app 選項 | `舊鞋救命`（debit 現金 + credit_kp 同額；Day3 達標再 credit_kp +50） |

**玩法步驟**
1. 學生決定奉獻金額，關主在 app 輸入金額 → 扣現金、入同額 KP。
2. 若單筆 ≥100 且該生本日未領過 bonus → 額外觸發 +50 KP。

**關主注意事項**
- bonus 每生限一次，後台計數，勿重複給。
- 單筆下限建議 10。

### 卡片 2-2：見證再開放（聽見證）

| 欄位 | 內容 |
|---|---|
| 關卡介紹 | Day3 可再聽不同同工的生命見證，規則同 Day2：每位同工對同一學生只給一次 **+100 KP**，找不同同工可累計。 |
| 開關詞 | 「想再聽一段見證嗎？換一位同工聊，再加 100 天國點數。今天是交帳日，能多聽就多聽。」 |
| 道具與數量 | 無 |
| 關主人數 | 各同工自行操作（建議 6–8 位同工，各自下載 app）|
| 準備時間 | 0 |
| 總時長（單次） | 2–5 分 |
| app 選項 | `分享見證`（credit_kp +100，固定值，無 user input）|

**關主注意事項**
- 同工 UID 綁定，同一同工對同一學生重複不再給。

### 卡片 2-3：回應／決志卡（牧養性質）

| 欄位 | 內容 |
|---|---|

## 3. 10:25 市場關閉突襲

### 卡片 3-1：市場關閉（主持＋總控）

| 欄位 | 內容 |
|---|---|
| 關卡介紹 | 結關前約 10 分，主持無預警喊停。所有攤位（含銀行）即刻關閉，學生手上**未兌換現金＋未提領定存本利一律 ×0.1**（銷毀 90%）。 |
| 開關詞 | （主持大聲）「停——！市場現在關閉！所有攤位停止交易，銀行也關了！你手上沒換成積分的現金，還有定存裡的錢，現在只剩十分之一！定存不是避風港，沒換的全部蒸發九成。」 |
| 道具與數量 | 哨子/麥克風 ×1 |
| 關主人數 | 1 人（主持）+ 1 人（app 總控）|
| 準備時間 | 0（突襲，不預告）|
| 總時長 | 即時喊停 + 約 1 分廣播 |
| app 選項 | 總控執行市場關閉（一鍵凍結；結算依 SOT §7.1/§7.4 自動 ×0.1）|

**操作步驟（app 一鍵關市場）**
1. 主持喊停的同時，總控在後台執行「市場關閉」→ 所有 stall/action 凍結，不能再 `debit`/`credit`/`deposit`/`withdraw`/`積分`兌換。
2. 結算公式（SOT §7.1）：
   - 總積分 = 已兌換積分 Σ + （手上現金 + 定存本利）× 0.1
3. 定存本利與手上現金一併視為「未兌換現金」，全部 ×0.1（SOT §7.4，封堵定存避險套利）。

**關主注意事項**
- 關閉前**絕不預告**，這是設計上的突襲（信息點：地上財寶帶不走）。
- 已在積分兌換攤鎖定的積分**不受 ×0.1 影響**，安全。
- 各攤關主聽到喊停立即停手，不再接受任何交易，包含正在排隊的學生。

## 4. 結關交帳（10:25–10:35）

### 卡片 4-1：交帳腳本（主持）

| 欄位 | 內容 |
|---|---|
| 關卡介紹 | 連結「大地主交帳」故事收尾：總資產與天國點數雙軌揭曉的張力，總資產第一 ≠ 神眼中的贏家。 |
| 開關詞 | 「故事裡的主人回來了，要跟每個僕人交帳。今天市場關了，你才發現：拼命賺的現金，沒換成積分就只剩十分之一；但天國點數——奉獻的、聽見證的、回應的——一點都沒少，全部帶得走。等一下發成績單，看看你這三天，把才幹用在哪裡。」 |
| 道具與數量 | 麥克風 ×1 |
| 關主人數 | 1 人（主持）|
| 準備時間 | 0 |
| 總時長 | 約 10 分 |
| app 選項 | 無（總控同步準備成績單列印）|

**交帳重點**
- 兩軌並行揭曉：**總積分前三名** + **天國點數前三名**。
- 獎品：娃娃（留 3 個）＋校園書房小禮物。
- 強調張力：及早把會朽壞的（現金）換成存得住的（積分／KP）。

## 6. 小組題（10:40–11:40）

1. 市場關閉那一刻，你手上的現金變成十分之一，你的第一個反應是什麼？這跟「地上的財寶」有什麼關係？
2. 你這三天賺最多的是用什麼方法？（公會、賭場、銀行…）回頭看，哪些是真正划算的？
3. 賭場三注全是負期望（房屋優勢 5.88% / 16.67%），你有沒有越想翻本越虧？這跟「貪心 vs 知足」說了什麼？
4. 天國點數（奉獻、聽見證、回應卡）是唯一帶得走的。對照成績單，你在「總資產」和「管家獎」兩軌的名次差多少？神看重哪一個？
5. 主人按才幹分派（才幹 5/2/1＝起始金 500/400/300），你拿到的起始金是多少？你把它用出來了，還是埋起來了？
