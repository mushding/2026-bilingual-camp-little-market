# 小市集營會 文件總目錄

> 主題「忠心好管家」：以三天的經濟大地遊戲，讓學員在賺取、消費、奉獻與投資中體會神所託付的管理之責。
> 全程以 NTAG 感應卡作為身分與錢包，刷卡完成攤位交易、記帳與兌換。
> 採「雙軌結算」——地上的金錢帳與天國的屬靈帳並行，最終一同回顧與結算。
> 本目錄收錄企劃設計與 App / Backend 技術設計，供企劃同工與實作 agent 共同參照。
