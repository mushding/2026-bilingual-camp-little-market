# 02 天國系統與最終結算

> 數字一律以 SOT v1.0 為唯一來源。幣別：「元」=遊戲現金、「積分」=最終資產分數、「KP」=天國點數（管家獎）。
> 核心張力：總資產第一 ≠ 神眼中的贏家。積分與 KP 為**完全獨立的兩條軌**。

---

## 1. 天國點數（KP）系統

KP **不可換回現金、不計入積分榜**，是獨立的第二軌「管家獎」。

### 1.1 所有來源與防刷上限

| 來源 | 給予方式 | 上限／防刷 | App |
|---|---|---|---|
| 奉獻（舊鞋救命） | 現金 1:1 → KP（奉 100＝−100 元、+100 KP） | 無硬上限，自限於現金；單筆下限建議 10 | `舊鞋救命` |
| 聊天聽見證 | +100 KP／次 | **同一同工對同一學生只給 1 次**；可找不同同工累計 | `分享見證` |
| 感謝卡（郵政核銷，加給**寄件人**） | +20 KP／張（每寄件人上限 3 張＝60 KP） | 在雜貨店買卡＝**花現金、不加 KP**；KP 改由**郵政同工**後台依卡上**寄件人名字**登記給寄件人；卡上**沒寫名字 → 不登記、不加 KP** | `郵政`（`mail_kp`，名字搜尋） |

防刷總則：
- 聽見證綁定同工 UID，重複同一同工不再給。
- 奉獻雖無硬上限，但消耗現金，自帶剎車。
- 感謝卡 KP 由**郵政同工**依卡上**寄件人名字**後台登記（每寄件人上限 3 張＝60 KP，`card_count` 計數）；買卡需花現金（自帶剎車）；卡上**沒寫名字無法登記**，務必到處提醒學生寄卡要寫自己名字。
- 同工人數＝聽見證 KP 天花板的實質決定者，建議 6–8 位（每位需各自下載 App）。

### 1.2 Day3 天國攤（沿用 Day2，不加開強化）

Day3 攤位＝Day2 全套，天國攤（奉獻、聽見證、感謝卡郵政核銷）與 KP 規則**完全沿用 Day2**，不另設強化天國橋段（**無**奉獻 bonus、**無**回應／決志卡、**無**見證再開放）。Day3 唯一差別是加強「最後一天交帳、天國點數帶得走」的曝光提醒（關主口頭與海報），不改任何 KP 數值或上限。

## 2. 雙軌結算

於 **D3 市場關閉時（10:25）** 同時結算兩軌。

### 2.1 積分榜（總資產軌）

```
總積分 = 已兌換積分 Σ
       + (手上現金 + 定存本利) × 0.1
```

- **已兌換積分**：在積分兌換攤已鎖定的分數，安全，不受 ×0.1 影響。
- **手上現金 + 定存本利**：市場關閉時所有未兌換現金，一律 ×0.1（90% 蒸發）。
- 策略涵義：聰明玩家在 10:25 前提領定存並兌換成積分；拖延者本利大半蒸發（信息點：地上財寶帶不走）。

### 2.2 管家獎（天國點數軌）

```
管家獎排名 = 總 KP 由高到低
```

獨立第二軌，與積分榜並行，刻意製造張力。

## 3. 銀行定存利息結算

| 項目 | 規則 |
|---|---|
| 利率 | **20%／天** |
| 結算時點 | **每場小市集結束結一次**（D1、D2、D3 各一次），**最多 3 次** |
| 計息方式 | **複利**，每場末對當前定存餘額 +20% |
| 進位 | 利息**無條件捨去到整數** |
| 提領 | 可隨時提領本利回到現金 |
| 造幣上限 | 最大 **+72.8%**（1.2³），構成硬天花板 |

**範例**：D1 定存 100，全程不動 →
- D1 末：120
- D2 末：144
- D3 末：172（100×1.2³＝172.8，捨去）

---

## 4. 市場關閉對結算的影響

- **10:25 主持無預警喊停**，所有攤位（含銀行）即刻關閉。
- 此刻起不能再兌換、不能再提領或操作定存。
- 手上現金與**定存本利一併視為未兌換現金**，全部 ×0.1 計入積分。
- 設計意圖：**定存不是避風港**，封堵「靠定存避險套利」的漏洞。

## 5. 最終計分公式與換算範例

### 5.1 積分兌換牌價

| 投入現金 | 換得積分 | 倍率 |
|---|---|---|
| 100 | 100 | 1.0 |
| 250 | 300 | 1.2 |
| 400 | 500 | 1.25 |
| 750 | 1000 | 1.33 |

- 可重複換、可組合。大額有獎，誘導「攢一波整批換」。
- 未兌換現金最終 ×0.1（90% 懲罰），積分則 1:1 甚至 1.25 保值 → 鼓勵及早把會朽壞的換成存得住的。

### 5.2 換算範例

換得 **1100 積分** ＋ 剩 **400 現金**：

```
最終總積分 = 1100 + 400 × 0.1 = 1100 + 40 = 1140 積分
```
